# 需求：流式识别（边说边出字）

## 背景

当前语音输入流程是"录完再识别"：按住热键录音 → 松开 → 整段音频送入 SenseVoice → 一次性输出结果。对于较长的语音输入，用户需要等到松开热键后才能看到任何文字，体验不够流畅。

## 需求描述

实现流式识别功能，让用户在按住热键说话的过程中就能看到识别结果逐步出现在焦点窗口中（边说边出字）。

## 具体要求

1. 录音过程中实时输出识别文字到焦点窗口
2. 输出的文字必须带标点符号（输入法基本要求）
3. 识别准确率不能显著低于当前 SenseVoice 离线模式
4. 支持中英文混合输入

## 结果

本轮开发未能成功实现，详见 SUMMARY.md。流式识别将在后续引入 Qwen3-ASR 时重新尝试。
