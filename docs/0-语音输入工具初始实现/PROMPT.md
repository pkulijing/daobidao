# 需求：Linux 语音输入工具

## 背景

用户在 Mac 上习惯使用微信输入法语音输入，手机上习惯使用豆包输入法语音输入，但在 Ubuntu 上没有好用的语音输入法工具。

## 需求描述

实现一个接近 Mac 微信输入法体验的 Linux 语音输入工具：

1. **任意输入框可用**：在任何应用的输入框中，按住键盘快捷键即可持续语音输入
2. **中英文混合**：支持中英文混合输入
3. **快捷键可配置**：默认使用右 Ctrl，支持切换为左右 Ctrl/Alt/Meta(Win)
4. **双引擎**：支持本地模型（SenseVoice-Small）和云端模型（豆包 ASR）切换

## 环境

- Ubuntu 24.04, X11
- GPU: RTX 4090 / RTX 3080（两台机器）
- 云端偏好：豆包（火山引擎）ASR
