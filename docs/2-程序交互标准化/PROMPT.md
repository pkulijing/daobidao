# 程序交互标准化

## 背景

whisper-input 目前是开发者工具形态：命令行启动、YAML 文件手动配置、setup.sh 安装。
对于普通用户来说门槛较高，需要标准化为 Ubuntu 桌面应用的交互方式。

## 需求

1. **图形化设置界面**：从系统托盘右键菜单打开设置页面（浏览器中），可以调整快捷键、语言、输入方式等配置
2. **DEB 包安装**：生成 .deb 文件，双击即可安装，自动处理系统依赖
3. **桌面菜单启动**：安装后在 Ubuntu 应用菜单中可见，点击即可启动
4. **开机自动启动**：支持 XDG autostart，在设置页面中通过开关控制
